# graphql
The objective of this project is to learn the GraphQL query language, by creating your own profile page.

## Run Locally

Clone the project

```bash
  git clone https://learn.zone01dakar.sn/git/papaabddiop/graphql.git
```

```bash
  cd graphql/
```

Start the server

```bash
  python3 -m http.server 
```
